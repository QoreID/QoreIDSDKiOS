QoreIDSDKiOS
QoreIDSDKiOS is the official iOS SDK for QOREID verification services, owned by VerifyMe Nigeria. This SDK simplifies user identity verification for iOS developers, supporting both Swift and Objective-C projects on iOS 11 and above.

Installation
To install the QoreID SDK in your iOS project, use CocoaPods:

Open your project’s Podfile.
Add QoreIDSDK to your project’s dependencies.
ruby
Copy code
pod 'QoreIDSDK'
Run the following command in your terminal:
bash
Copy code
pod install
Open the .xcworkspace file generated by CocoaPods, and you’re ready to start using the QoreID SDK in your project.
Requirements
iOS 11 or higher.
Xcode 10.1 or later.
Compatible with Swift and Objective-C projects.
