# QoreIDSDKiOS

**QoreIDSDKiOS** is the official iOS SDK for QOREID verification services, owned by VerifyMe. This SDK simplifies user identity verification for iOS developers, supporting both Swift and Objective-C projects on iOS 11 and above.

## Installation

To install the QoreID SDK in your iOS project, use CocoaPods:

1. Open your project’s `Podfile`.
2. Add `QoreIDSDK` to your project’s dependencies.

   ```ruby
   pod 'QoreIDSDK'
