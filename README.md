# QoreIDSDK

**QoreIDSDK iOS ** is the official iOS SDK for QOREID verification services, owned by VerifyMe.

## Installation

To install the QoreID SDK in your iOS project, use CocoaPods:

1. Open your project’s `Podfile`.
2. Add `QoreIDSDK` to your project’s dependencies.

   ```ruby
   pod 'QoreIDSDK'

3. Run the following command in your terminal:
   ```ruby
   pod install
4. Requirements
   iOS 11 or higher.
   Xcode 10.1 or later.
   Compatible with Swift and Objective-C projects.

## Usage
Importing the SDK
After installation, import the SDK in your code:

Swift
  ```ruby
   import QoreIDSDK
