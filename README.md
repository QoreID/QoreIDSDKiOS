# QoreIDSDKiOS

**QoreIDSDKiOS** is the official iOS SDK for QOREID verification services, owned by VerifyMe.

## Installation

To install the QoreID SDK in your iOS project, use CocoaPods:

1. Open your project’s `Podfile`.
2. Add `QoreIDSDK` to your project’s dependencies.

   ```ruby
   pod 'QoreIDSDK'
